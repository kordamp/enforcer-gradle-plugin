
= AlwaysFail

This rule will always fail. It is useful for testing proper plugin configuration.

== Allowed Phases
* BEFORE_BUILD
* BEFORE_PROJECTS
* BEFORE_PROJECT
* AFTER_PROJECT
* AFTER_PROJECTS
* AFTER_BUILD

== Default Phases
* BEFORE_BUILD
* BEFORE_PROJECTS
* BEFORE_PROJECT
* AFTER_PROJECT
* AFTER_PROJECTS
* AFTER_BUILD

== Configuration
[source,groovy]
[subs="+macros"]
----
enforce {
    rule(enforcer.rules.AlwaysFail) { r ->
        r.enabled
        r.enforcerLevel
        r.message
        r.phases
    }
}
----
